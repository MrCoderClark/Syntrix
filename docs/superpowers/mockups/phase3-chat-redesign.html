Mike Chen Yesterday at 10:28 AM

Yeah! We just adopted the iter pattern for a streaming data pipeline. Works really well with slices.Collect. Here's a rough example:

func FilterUsers(users iter.Seq[User]) iter.Seq[User] { return func(yield func(User) bool) { for u := range users { if u.Active { if !yield(u) { return } } } } }

The lazy evaluation is the real win — no intermediate slices.
